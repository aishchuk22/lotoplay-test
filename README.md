# 🎸 Гурд «Грим та Грім» — Офіційний сайт

Сучасний лендінг для рок-гурту, створений з фокусом на продуктивність, адаптивність та драйвовий дизайн. Проєкт демонструє поєднання естетики важкої музики та передових вебтехнологій.

## 🚀 Стек технологій

- **HTML** — семантична та доступна розмітка для кращого SEO.
- **CSS** — кастомні гріди (Grid), гнучкі флекси (Flexbox) та складні градієнти для створення атмосфери глибини.
- **JavaScript** — логіка модальних вікон (Pop-up), плавна анімація інтерфейсу та обробка даних форми.

---

## 🛠 Використані інструменти та сервіси

### 🎨 Контент та ШІ

- **ChatGPT / Gemini** — генерація унікальних концепт-артів для учасників гурту та допомога в написанні текстів.
- **Copilot** — допомога в написанні JavaScript-коду.

### ⚡ Оптимізація та Швидкість

- **Squoosh** — стиснення зображень без втрати візуальної якості.
- **Convertico** — конвертація всієї графіки у формат **AVIF**, що дозволило зменшити вагу сторінки в кілька разів.
- **PageSpeed Insights** — основний інструмент для аудиту продуктивності та досягнення максимальних показників завантаження.
